This project was started as a university project at
the chair for multimedia computing (www.multimedia-computing.org)
at Augsburg University and created by Christoph Lassner
(www.multimedia-computing.org/wiki/Christoph_Lassner).
The supervisors were Prof. Rainer Lienhart and Thomas Greif.

The first version was released in June 2011.

and a detailed documentation, see the attached README.PDF.
